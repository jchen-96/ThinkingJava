#### 15.2.1 一个元组类库
元组的概念：它是将一组对象直接打包存储于其中一个单一对象。这个容器对象允许读取其中的元素，但是不允许向其中存放新的对象。（这个概念也称为数据传送对象，或信使）。

<font color='red'>对于元组中的元素，我们没必要将其声明为private 可以使用final关键字，达到只读的效果。</font>
